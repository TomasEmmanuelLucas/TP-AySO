## Acerca del Proyecto

El presente trabajo tiene como objetivo aplicar conceptos de virtualización mediante la instalación y uso de una máquina virtual con sistema operativo Ubuntu, utilizando Oracle VirtualBox. Dentro de este entorno virtual se desarrolló y ejecutó un programa en Python, con el fin de simular un entorno de desarrollo controlado y validar su funcionamiento en un sistema aislado.


## Tecnologías Utilizadas

* [Python](https://www.python.org/)
* [VirtualBox](https://www.virtualbox.org/)
* [Ubuntu](https://ubuntu.com/)
